Project Description:
This project is a console-based Tic Tac Toe game developed using C++. It allows a player to
compete against a computer using basic AI logic. The aim was to understand programming
fundamentals through a practical application.
Objectives:- Build a working game using C++- Apply programming concepts like loops and conditions- Implement basic AI logic
Features:- Player vs Computer gameplay- Win and draw detection- Simple AI (win/block strategy)
Technical Approach:
The game uses a 3x3 array to represent the board. Functions are used to display the board, check
for wins, and handle moves. The AI checks for winning or blocking moves before choosing a
random position.
Challenges:
Some challenges included handling invalid input and correctly implementing the AI logic. These
were solved through testing and debugging.
